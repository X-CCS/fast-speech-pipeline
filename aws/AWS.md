## Useful Commands

Activate PyTorch Environment:
`source activate pytorch_p36`

SSH:
`ssh -i aws/wiki-podcast.pem ubuntu@[IP]`

SCP:
`scp -i aws/wiki-podcast.pem path/to/file ubuntu@[IP]:~`
